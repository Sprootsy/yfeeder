Edit is now open source

The article announces that the source code for the classic MS-DOS text editor, EDIT.COM, is now open source and available on GitHub under the MIT license. This marks a significant event for those familiar with the early days of DOS and Windows, as EDIT was a ubiquitous tool for creating and modifying text files. The release includes the original source code, build tools, and instructions for compiling the editor. The post also briefly touches on the history of EDIT and its role in the evolution of text editing on PCs. The purpose of releasing the source code is primarily for educational and historical preservation, allowing developers and enthusiasts to explore and learn from the codebase of this vintage application.

Key Points:

  • The source code for MS-DOS EDIT.COM is now open source.
  • It is available on GitHub under the MIT license.
  • The release includes the original source code, build tools, and compilation instructions.
  • The primary goal is for educational and historical preservation purposes.
  • EDIT was a widely used text editor in the MS-DOS and early Windows environments.
