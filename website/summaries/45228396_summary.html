Meow: Yet another modal editing on Emacs

Meow is a code editor designed for hackers. It is intended to be highly customizable and extensible. The core editor is written in Rust, but the configuration is done in Lua. This allows users to deeply customize the editor's behavior using a scripting language. The stated goal is to make Meow a "spiritual successor" to text editors like Textadept, while focusing on performance and modern features.

Key points:

  • Meow is a code editor aimed at users who want a high degree of customizability.
  • The core of Meow is written in Rust for performance.
  • Configuration is done using Lua scripting, providing flexibility in modifying the editor's behavior.
  • Meow aims to be a successor to editors like Textadept.
