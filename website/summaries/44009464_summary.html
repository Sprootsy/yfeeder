ClojureScript 1.12.42

The ClojureScript team has announced the release of ClojureScript version 1.12. It's the first release in over a year, and includes numerous new features, improvements, and bug fixes. The new release focuses on developer experience and compiler performance. Key new features include support for ES2022 features like class static initialization blocks and top-level await, better support for npm modules including direct reference to npm modules from ClojureScript, enhanced source maps for better debugging, and improved compiler performance. In addition, the release provides improvements to interop with JavaScript libraries, enhanced macro support, and expanded tools for advanced compilation. Several bug fixes are also included. The ClojureScript team indicates that this release will be a significant step forward for ClojureScript developers, providing increased productivity and flexibility.
