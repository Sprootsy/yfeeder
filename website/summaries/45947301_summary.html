Z3 API in Python: From Sudoku to N-Queens in Under 20 Lines

This article is a tutorial on how to use Z3, a theorem prover from Microsoft Research, with its Python bindings (Z3Py). It covers various aspects of using Z3 to solve constraints and model problems.

The tutorial starts by introducing the basic concepts of Z3, such as declaring variables of different types (integers, reals, Booleans, bit-vectors, arrays, datatypes) and creating constraints using logical operators. It explains how to solve these constraints using the `solve()` function and how to check the satisfiability of a set of constraints. The article demonstrates how to retrieve a model when a solution is found, allowing you to inspect the values assigned to the variables.

The article explains how to simplify and display the results.

Several examples are presented to illustrate the use of Z3 in different scenarios. These examples include:

  • Solving simple arithmetic and Boolean puzzles.
  • Finding solutions to systems of equations.
  • Using quantifiers to express properties over ranges of values.
  • Modeling and solving Sudoku puzzles.
  • Solving the N-Queens problem.
  • Solving the Water Jug problem, demonstrating the use of arrays to model state.
  • Using datatypes to model lists and other recursive structures, including defining functions to operate on those datatypes.
  • Defining recursive functions and proving properties about them.

The tutorial also demonstrates the use of tactics and solvers in Z3. Tactics are simplification and solving strategies that can be applied to a goal. Solvers are the engines that actually find solutions to the constraints.

Furthermore, the article touches upon advanced topics such as:

  • Using Z3 to verify software.
  • Finding errors in programs.

Key Points:

  • Z3 is a theorem prover that can be used to solve constraints and model problems.
  • Z3Py is the Python bindings for Z3, providing a convenient way to interact with Z3.
  • Variables can be declared with different types (integers, reals, Booleans, etc.).
  • Constraints are created using logical operators.
  • The `solve()` function is used to solve constraints.
  • Models can be retrieved to inspect the values assigned to variables in a solution.
  • Quantifiers can be used to express properties over ranges of values.
  • Arrays can be used to model state in problems.
  • Datatypes can be used to model lists and other recursive structures.
  • Recursive functions can be defined and properties about them can be proven using Z3.
  • Tactics and solvers can be used to control the solving process.
  • Z3 can be used to verify software and find errors in programs.
