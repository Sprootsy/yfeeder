Leaked Apple M5 9 core Geekbench scores

This article is a benchmark result from Geekbench 6, specifically for a CPU test. The device tested is an Apple MacBookPro16,1, equipped with an Intel Core i9-9880H processor. This processor has 8 cores and 16 threads, with a base clock speed of 2.30 GHz. The test was conducted on macOS 14.4.1.

The Geekbench 6 CPU benchmark suite measures the processor's performance in both single-core and multi-core workloads. The MacBook Pro achieved a single-core score of 1592 and a multi-core score of 7233.

The details provided also include information about the Geekbench version used (6.2.0) and the libraries used for the test (version 6.2.0).

Key Points:

  • Device: Apple MacBookPro16,1
  • Processor: Intel Core i9-9880H (8 cores, 16 threads, 2.30 GHz base clock)
  • Operating System: macOS 14.4.1
  • Geekbench Version: 6.2.0

  • Multi-Core Score: 7233
