Short Description:

Bluetui is a terminal-based application that simplifies Bluetooth device management, particularly for BLE devices. Built using Python, it provides a user-friendly text-based interface for scanning, connecting to, and interacting with Bluetooth devices directly from the command line. Bluetui addresses the need for a convenient and efficient way to manage Bluetooth connections without relying on graphical user interfaces or complex command-line tools. It is built on the Bleak library, ensuring compatibility with BLE devices, and utilizes the textual framework for creating an intuitive and interactive TUI. Bluetui aims to be a practical tool for developers, hobbyists, and anyone who frequently works with Bluetooth devices in a Linux environment.
