IRS open sources its fact graph

The article is a repository on GitHub for the IRS's Fact Graph project. The Fact Graph is a knowledge graph that contains information extracted from various IRS forms and publications. The goal of the Fact Graph is to improve the IRS's ability to answer questions, detect fraud, and improve compliance. The repository contains the schema for the Fact Graph, as well as code for extracting and loading data into the graph.

Key points:

  • The IRS Fact Graph is a knowledge graph.
  • It is built from IRS forms and publications.
  • Its purpose is to answer questions, detect fraud, and improve compliance.
  • The repository contains the schema and code for building the graph.
