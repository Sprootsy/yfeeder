OpenBSD 7.8

OpenBSD 7.8 has been released. It features improved hardware support, performance enhancements, and security improvements.

Key Points:

  • New Hardware Support: Improved support for various hardware platforms including amd64, arm64, armv7, i386, macppc, powerpc64, and riscv64. Specific improvements are noted for Intel, AMD, and ARM processors and graphics.
  • Driver Improvements: Enhanced drivers for network interfaces, storage controllers, and USB devices.
  • Kernel Improvements: Various kernel-level improvements enhancing overall system stability and performance.
  • Security Improvements: Updates to OpenSSH, LibreSSL, and other security-related components to address vulnerabilities and improve security posture.
  • Networking Improvements: Enhancements to network stack and related tools.
  • Installer Improvements: Improvements and bug fixes in the installation process.
  • Software Updates: Updated versions of many included software packages.
