S&box is now an open source game engine

Update 25 is a substantial update to sbox, focusing on improving the user experience, adding new features, and addressing bugs. A key focus is the revamp of the asset browser, now known as the "Inventory," aiming to provide a more intuitive and efficient way for users to manage and utilize their assets. This includes improved search, filtering, and organization options. The update also introduces significant enhancements to the asset pipeline, streamlining the process of importing, creating, and modifying assets.

A notable addition is the introduction of "Smart Snapping," which allows for more precise and intelligent object placement within the game world. This feature helps users align objects more easily and create more polished and professional-looking environments.

The update brings several new assets to the game, expanding the available content for creators. These include a variety of props, models, and materials that can be used to enhance the visual appeal and functionality of user-created experiences.

Furthermore, Update 25 includes various bug fixes and performance improvements. These fixes address issues reported by the community and aim to enhance the overall stability and smoothness of the game. Performance optimizations are also included, targeting improved frame rates and reduced loading times, particularly in complex scenes.

The update also touches on the user interface, refining certain elements to provide a clearer and more user-friendly experience. These UI improvements aim to make it easier for users to navigate the game's various tools and features.

Key Points:

  • Inventory Revamp: Overhauled asset browser with improved search, filtering, and organization.
  • Asset Pipeline Enhancements: Streamlined asset creation, import, and modification process.
  • Smart Snapping: New feature for precise and intelligent object placement.
  • New Assets: Introduction of new props, models, and materials.
  • Bug Fixes: Addressed various issues reported by the community.
  • Performance Improvements: Optimizations for better frame rates and reduced loading times.
  • UI Improvements: Refined user interface elements for a clearer and more user-friendly experience.
