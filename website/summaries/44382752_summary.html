This commit introduces a new CPU model, "Intel Sapphire Rapids (SPR) X86", to the QEMU emulator. This model incorporates several new features and architectural updates found in Intel's Sapphire Rapids processors. The primary updates include:

  • AVX-512 FP16 support: Adds support for the AVX-512 FP16 instruction set, enabling efficient processing of half-precision floating-point numbers.
  • Advanced Matrix Extensions (AMX): Implements support for AMX, which includes:
    • Tile architecture: Provides a tile-based architecture for matrix operations.
    • Tile configuration registers (TMM): Includes registers for configuring tiles and managing matrix operations.
    • New instructions: Introduces new instructions specific to AMX, enhancing matrix computation capabilities.
  • User-Mode Interrupts (UMI): Introduces User-Mode Interrupts, which allow user-level applications to handle interrupts directly.
  • SGX modifications: Updates related to Software Guard Extensions (SGX), aligning with the Sapphire Rapids architecture.

The commit details the necessary code modifications to incorporate these features into the QEMU x86 emulation environment. This includes:

  • Adding new CPU definitions and feature flags.
  • Implementing the behavior of new instructions.
  • Updating relevant control and status registers.
  • Modifying the instruction decoding logic to recognize and handle the new instructions.

The overall goal of this commit is to provide a more accurate and feature-complete emulation of Intel Sapphire Rapids processors within QEMU, enabling developers to test and optimize software for this architecture. The changes ensure that QEMU can correctly execute code that utilizes the new features available in Sapphire Rapids, such as AVX-512 FP16 and AMX.
