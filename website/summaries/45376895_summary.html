Redox OS Development Priorities for 2025/26

Summary:

The Redox OS development priorities for the upcoming year focus on enhancing the operating system's usability, compatibility, and overall functionality. Key areas of focus include improving driver support, particularly for graphics and network devices, to broaden hardware compatibility. The developers aim to refine the user interface and desktop environment, likely through improvements to Orbital, the windowing system, to provide a more polished and intuitive user experience.

A significant effort will be dedicated to enhancing Redox's POSIX compatibility, allowing a wider range of existing software to run on the OS. This includes improving the implementation of system calls and standard libraries. The development team also plans to strengthen the security model of Redox OS by implementing and refining capabilities-based security features, aiming for a more secure and robust system.

Further work is planned on the file system, likely focusing on stability, performance, and potentially adding features. The goal is to make Redox a more viable option for a wider range of users and use cases.

Key Points:

  • Improved driver support, especially for graphics and network devices.
  • Refinement of the user interface and desktop environment (Orbital).
  • Enhanced POSIX compatibility.
  • Strengthening the security model with capabilities-based security.
  • File system improvements focusing on stability and performance.
