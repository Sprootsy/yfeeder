$2 WeAct Display FS adds a 0.96-inch USB information display to your computer

The article discusses the 2-WeAct Display FS, a small USB information display. It is designed to be connected to a computer and show system information like CPU usage, RAM usage, disk usage, network speed, and temperature. The device features a 0.96-inch LCD screen and is housed in an aluminum alloy enclosure. The display is powered and communicates with the host computer via a USB Type-C connection.

The display is compatible with Windows, Linux, and macOS. Users can customize what information is displayed and how it is presented via configuration files. The software and drivers are available on GitHub.
