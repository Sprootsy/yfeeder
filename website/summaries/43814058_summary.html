OpenBSD 7.7 Released

OpenBSD 7.7 was released on May 16, 2024. This release includes various improvements and updates across the system.

In terms of installation and upgrade, the release supports various platforms including amd64, arm64, i386, macppc, riscv64, and sparc64. Users can upgrade via the sysupgrade utility or manual upgrade process.

Several kernel improvements are included such as enhanced hardware support, updates to drivers, and bug fixes. Specifically, there are updates for the amd64, arm64, and riscv64 architectures including mitigations for hardware vulnerabilities.

Networking has seen significant updates with improvements to the TCP/IP stack, wireless networking, and VPN functionalities. The relayd(8) daemon has also been enhanced.

The base system has been updated with new versions of various tools and libraries, including OpenSSH 9.7, LibreSSL 3.9.1, and Go 1.22.2. Additionally, there are improvements to system utilities and libraries.

Numerous drivers have been added or improved. These include updates for audio, network, storage, and video devices, enhancing compatibility with a wide range of hardware.

Key Points:

  • OpenBSD 7.7 released on May 16, 2024.
  • Supports multiple platforms: amd64, arm64, i386, macppc, riscv64, sparc64.
  • Upgrade via sysupgrade or manual process.
  • Kernel improvements for hardware support and security.
  • Networking enhancements including TCP/IP, wireless, and VPN.
  • Updated base system tools: OpenSSH 9.7, LibreSSL 3.9.1, Go 1.22.2.
  • Driver updates for audio, network, storage, and video devices.
