The article describes oxdraw, a Python package designed for creating publication-quality figures for crystallography and solid-state chemistry. It leverages the power of Matplotlib while simplifying the process of generating complex visualizations, like crystal structures, electron densities, and reciprocal space representations. The package aims to streamline the figure creation workflow, allowing researchers to produce aesthetically pleasing and informative graphics with minimal coding effort. It offers a high-level interface for customizing figure appearance and supports various file formats for exporting the created images.
