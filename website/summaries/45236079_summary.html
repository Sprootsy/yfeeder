Pass: Unix Password Manager

The article describes pass, the standard Unix password manager. Passwords are stored in GPG encrypted files organized in a directory tree. Pass can generate, store, edit, and retrieve passwords from the command line. It can also copy passwords to the clipboard and supports version control using git. The password store is typically located in ~/.password-store.

Key Points:

  • Pass is a command-line password manager for Unix-like operating systems.
  • Passwords are stored in GPG-encrypted files.
  • Passwords are organized in a directory tree structure.
  • Pass can generate new passwords.
  • Pass can store existing passwords.
  • Pass can edit existing passwords.
  • Pass can retrieve passwords.
  • Passwords can be copied to the clipboard.
  • Version control is supported via git.
  • The default password store location is ~/.password-store.
