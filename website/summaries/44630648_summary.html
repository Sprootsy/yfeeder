Show HN: X11 desktop widget that shows location of your network peers on a map

Connmap is a command-line tool and Go library for visualizing network connections. It captures local socket information, including process details, and maps connections to GeoIP locations. The tool supports filtering connections by various criteria such as ports, processes, and IP addresses. It can output data in JSON, CSV, or graph formats. Connmap aims to provide a clear, visual representation of network activity, aiding in network analysis, security monitoring, and troubleshooting.
