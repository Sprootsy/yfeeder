F2 is a command-line tool designed for batch renaming files and directories quickly and safely. It offers a wide range of renaming options using a simple and intuitive syntax. The tool emphasizes safety by providing a preview of changes before they are applied and includes features to prevent common errors, such as overwriting existing files. F2 supports various renaming schemes, including sequential renaming, using regular expressions, and modifying parts of filenames.

Key features and functionalities include:

  • Batch Renaming: Enables renaming multiple files and directories in a single command.
  • Regular Expression Support: Allows complex renaming operations using regular expressions for pattern matching and replacement.
  • Sequential Renaming: Facilitates renaming files with sequential numbers, useful for organizing large sets of files.
  • Find and Replace: Provides a straightforward way to find specific strings in filenames and replace them with new strings.
  • Undo Functionality: Offers an undo option to revert the last renaming operation, enhancing safety.
  • Conflict Detection: Detects potential conflicts (e.g., overwriting files) before renaming and provides options to resolve them.
  • Dry Run: Allows users to preview the renaming changes without actually applying them.
  • Cross-Platform Compatibility: Works on various operating systems, including Windows, macOS, and Linux.
  • Metadata Integration: Can use file metadata (e.g., creation date, modification date) in the renaming process.
  • Customizable Formatting: Supports customizable formatting options for renaming, such as changing case (uppercase, lowercase).

The tool's design focuses on user-friendliness, with clear error messages and comprehensive documentation to guide users through different renaming scenarios. It also emphasizes performance, being optimized to handle large numbers of files efficiently. The tool is open-source, allowing community contributions and customizations.
