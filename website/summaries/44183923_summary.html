Apple Notes Expected to Gain Markdown Support in iOS 26

Based on rumors, the Apple Notes app may gain Markdown support in iOS 26, expected to be released in 2025. While details are scarce, this addition would allow users to format text using Markdown syntax, a popular and simple way to add styling like headings, lists, and emphasis to plain text. This feature would bring Apple Notes more in line with other note-taking apps that already offer Markdown support, potentially making it a more attractive option for users who prefer this formatting method.

Key Points:

  • Apple Notes is rumored to get Markdown support.
  • The feature is expected to arrive with iOS 26 in 2025.
  • Markdown support would allow users to format text using Markdown syntax.
