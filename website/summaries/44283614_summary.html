Simplest C++ Callback, from SumatraPDF

The article discusses how to implement a simple C callback function from SumatraPDF, a PDF viewer. It focuses on creating a basic plugin that allows SumatraPDF to call a C function when a document is loaded. The author emphasizes simplicity and aims to provide a clear and straightforward example for developers who want to extend SumatraPDF's functionality. The blog post walks through the necessary steps, from setting up the development environment to writing the C code and integrating it with SumatraPDF. The core of the process involves creating a DLL (Dynamic Link Library) with an exported function that SumatraPDF can call. The DLL needs to be placed in SumatraPDF's plugin directory. When SumatraPDF loads, it detects the DLL and attempts to call the exported function. The blog post covers creating a simple DLL in C that displays a message box when called by SumatraPDF. It covers basic C code, including the `windows.h` header, which is needed for creating a Windows DLL. The article specifically details how to define the exported function using `__declspec(dllexport)` so SumatraPDF can find it. It also shows how to create a `.def` file to define the exported function. The article explains the rationale for defining the exported function.
