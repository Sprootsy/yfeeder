Qt Creator 18 Released

Qt Creator 18 has been released with a primary focus on integrating new language server features and improving overall stability. The update incorporates features contributed by the community and the Qt team.

Key Features and Improvements

  • Language Server Protocol (LSP) Enhancements:
    • Qt Creator 18 enhances the integration of Language Server Protocol (LSP) clients, offering a more refined user experience.
    • The update includes options to set environment variables for LSP servers and allows disabling formatting for specific file types.
  • C++ Support:
    • There is a new option to prevent auto-activation of Clangd for very large projects to avoid performance issues.
  • QML Support:
    • Adds experimental support for rename refactoring.
    • Adds experimental support for code actions, like adding missing property defaults.
    • The update provides improved Quick fixes and code completion.
  • CMake Integration:
    • The update introduces better integration for CMake projects.
  • Debugging:
    • The integrated debugger has seen improvements, offering a more seamless debugging process.
  • User Interface and Experience:
    • The user interface has been refined, contributing to an improved user experience.
  • General Improvements:
    • The release includes numerous bug fixes and performance improvements to ensure a smoother and more efficient development workflow.

Key Points:

  • Qt Creator 18 focuses on enhancing language server integration and improving stability.
  • LSP integration is improved with options for environment variables and disabling formatting for certain file types.
  • C++ support is updated with a new option to prevent auto-activation of Clangd for very large projects.
  • Includes experimental support for rename refactoring and code actions in QML.
  • The update provides improved Quick fixes and code completion in QML.
  • The update offers better integration for CMake projects.
  • The debugger has been improved.
  • Numerous bug fixes and performance improvements are included.
