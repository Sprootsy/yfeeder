7-Zip 25.00

The article is a release note for 7-Zip version 25.00, released on 2024-04-21. The key changes and improvements in this version include:

  • New switches -myu and -myv to specify user name and group name for tar archives on UNIX-like systems.
  • The command line version now supports long path names (more than 260 characters) in Windows.
  • A bug in 7-Zip File Manager, which caused incorrect file dates when dragging files to some folders, has been fixed.
  • A new option "Extract here" has been added to the context menu for archives in 7-Zip File Manager.
  • Several bugs were fixed.
