Stuffed-Na(a)N: stuff your NaNs

The GitHub repository si14/stuffed-naan-js is a JavaScript project that aims to provide a simple, embeddable editor. The editor is implemented in pure JavaScript and has no external dependencies. The project is contained within a single HTML file making it easily integratable. The editor has undo/redo functionality.

Key points:

  • JavaScript project
  • Simple embeddable editor
  • Pure JavaScript
  • No dependencies
  • Single HTML file
  • Undo/redo functionality
