LineageOS 23

The LineageOS blog post announces the release of LineageOS 21, based on Android 14. The post details the changes and improvements included in this major update, as well as highlighting changes from previous versions.

Key Points:

  • Android 14 Foundation: LineageOS 21 is based on the Android 14 release.
  • WebView Updates: Updated WebView to Chromium 120.0.6099.144.
  • Build Infrastructure Improvements: Significant improvements were made to the build infrastructure.
  • Security Patch Updates: Merged Android’s December 2023 security patches.
  • Revamped Updater App: The Updater app has been largely re-written to align with the new Android features.
  • Android TV builds: Android TV builds now ship with an ad-free leanback launcher.
  • Optional App Signing: Added optional support for signing builds with your own keys.
  • AOSP Gallery: The default camera app, Aperture, now supports Android’s new Ultra HDR format.
  • Customization: Customization settings have been moved to a dedicated section in Settings.
