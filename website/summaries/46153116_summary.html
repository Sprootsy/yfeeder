Django 6

Summary

Django 6.0 is a feature release that introduces several new capabilities and improvements while also dropping support for some older components. This release supports Python versions 3.10, 3.11, and 3.12. Support for Python 3.9 has been removed.

Key new features and changes include:

  • Asynchronous Capabilities: Enhancements to asynchronous support are a major theme in Django 6.0. Specifically, the release adds the ability to use asynchronous template rendering.
  • New Template Engine Features: The Django template engine now supports asynchronous rendering, which could potentially improve the performance of template rendering in asynchronous views.
  • Form Rendering Improvements: The rendering of forms has been improved.
  • Changes to Field Lookups: There have been some changes to field lookups.
  • Removal of Backends: The Spatialite backend has been removed.
  • Dropped Support: Django 6.0 drops support for Python 3.9.

Key Points

  • Django 6.0 is a feature release.
  • Supported Python versions: 3.10, 3.11, and 3.12.
  • Python 3.9 support has been removed.
  • Asynchronous template rendering is now supported.
  • The Spatialite backend has been removed.
  • Form rendering has been improved.
  • Changes to field lookups.
