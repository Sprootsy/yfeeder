Summary

The article "Ant and Apples" presents a mathematical puzzle involving an ant traversing the edges of an equilateral triangle. The triangle's vertices are labeled A, B, and C, and an apple is placed at each vertex. The ant starts at vertex A and moves randomly between adjacent vertices (either clockwise or counterclockwise). The goal is to determine the expected number of moves the ant will take to eat all the apples. Eating an apple occurs when the ant visits the vertex where the apple is located.

The author begins by framing the problem as a Markov chain, where the state of the system is defined by the set of apples the ant has already eaten. There are eight possible states, ranging from having eaten no apples (the initial state) to having eaten all three apples. The article highlights the symmetry in the problem. For instance, the expected number of moves to eat the first apple is relatively straightforward to calculate, since the ant starts at vertex A where an apple is located.

The core of the article involves setting up and solving a system of linear equations to determine the expected number of moves to reach the absorbing state (all apples eaten) from each of the other possible states. The author carefully explains the reasoning behind each equation, leveraging the concept of conditional expectation. For example, the expected number of moves to eat all apples starting from the state where only one apple has been eaten is equal to 1 (for the next move) plus the average of the expected number of moves from the two possible states the ant could transition to after that move.

The symmetry of the triangle is used to simplify the problem. Since the triangle is equilateral, the expected number of moves to eat the second apple, starting from having eaten only one, is the same regardless of which apple was eaten first. Similarly, when two apples have been eaten, the expected number of additional moves to eat the last apple is the same regardless of which two apples were eaten.

The author then presents the system of equations in matrix form, which allows for a more compact representation and facilitates solving the system using computational tools. The solution to the system of equations yields the expected number of moves to eat all three apples starting from the initial state (no apples eaten).

Finally, the article discusses a slightly modified version of the problem where the ant is allowed to stay at the same vertex with a certain probability on each move. The author demonstrates how the system of equations needs to be adjusted to account for this new possibility. The article concludes by emphasizing the power of using Markov chains and systems of linear equations to solve problems involving expected values and random processes.
