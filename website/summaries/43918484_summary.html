Ty: A fast Python type checker and language server

Ty is a work-in-progress static type checker for Python, intended to be fast and easy to integrate. It's developed by Astral, the same team behind Ruff. Ty aims to address the slow performance and complex integration often associated with existing Python type checkers like mypy.

The stated goals of Ty are:

  • Fast type checking: Ty is built for speed, aiming to provide quick feedback during development.
  • Easy integration: The developers want Ty to be readily integrated into existing workflows and tools.
  • Modern architecture: Ty is built with a modern architecture for better performance and maintainability.

The repository's README provides basic usage examples. Ty can be run on a Python file using the command `ty check .py`. It will then analyze the file and report any type errors it finds.

The project is under active development, and contributions are welcome. The repository includes information on how to build and run Ty locally, as well as guidelines for contributing code.

Key Points:

  • Ty is a new static type checker for Python.
  • It is being developed by Astral, the creators of Ruff.
  • It focuses on speed, easy integration, and a modern architecture.
  • It is still a work-in-progress.
  • Contributions are welcome.
