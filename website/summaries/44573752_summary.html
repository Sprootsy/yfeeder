Plasma Bigscreen rises from the dead with a better UI

Plasma Bigscreen, KDE's open-source Android TV alternative, has been revived with a revamped user interface. Initially introduced in 2019, the project aims to provide a customizable and privacy-focused smart TV experience. The updated version features a redesigned UI based on KDE's MauiKit, offering improved navigation and a more modern look. The new interface is built with convergent apps in mind, meaning the apps can be used in both desktop and TV environments. The updated Plasma Bigscreen features a new home screen, an improved app launcher, and better support for voice control. The new UI is designed to be more intuitive and user-friendly. The developers plan to release the updated version soon.
