Show HN: Tkintergalactic - Declarative Tcl/Tk UI Library for Python

The article describes "tkintergalactic," a Python package that provides a customizable 2D space simulator using Tkinter. It enables users to create and simulate gravitational interactions between celestial bodies, offering features such as adding planets, stars, and black holes with adjustable properties like mass, velocity, and radius. The simulator allows users to observe the orbital mechanics and effects of gravity in a visually appealing environment. Additionally, it allows users to adjust the time scale, toggle trails to track the movement of celestial bodies, and display velocity vectors.

Key Points:

  • tkintergalactic: A Python package for simulating 2D space and gravitational interactions using Tkinter.
  • Celestial Bodies: Supports adding planets, stars, and black holes.
  • Customizable Properties: Allows users to adjust the mass, velocity, and radius of celestial bodies.
  • Visual Simulation: Provides a visual representation of orbital mechanics and gravitational effects.
  • Adjustable Time Scale: Users can control the simulation speed.
  • Trails: Option to display trails to visualize the movement of celestial bodies.
  • Velocity Vectors: Option to display velocity vectors.
