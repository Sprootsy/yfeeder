Zed Debugger

The article introduces Zed's new debugger, highlighting its seamless integration with the editor and its features aimed at streamlining the debugging process. It emphasizes the debugger's ability to automatically connect to running processes, its inline value display, and its support for various languages like Go, Node.js, Python, and Rust. The debugger also allows users to set breakpoints, step through code, inspect variables, and evaluate expressions, all within the Zed editor. The article also mentions future plans for the debugger, including support for more languages and debugging scenarios.

Key Points:

  • Zed now has a built-in debugger.
  • The debugger is seamlessly integrated into the Zed editor.
  • It automatically connects to running processes.
  • Inline value display shows variable values directly in the editor.
  • Supported languages include Go, Node.js, Python, and Rust.
  • Users can set breakpoints, step through code, and inspect variables.
  • Expression evaluation is supported.
  • Future plans include support for more languages and debugging scenarios.
