CPU-X: CPU-Z for Linux

Summary

CPU-X is a free and open-source software application designed to provide detailed information about a computer's CPU, motherboard, and other hardware components. It's similar to CPU-Z but aims to be cross-platform, supporting Windows, Linux, macOS, and FreeBSD. The software presents a user-friendly interface that displays specifications such as the processor's name, vendor, architecture, clock speeds, cache sizes, and supported instructions. For the motherboard, it shows the manufacturer, model, chipset, and BIOS version. CPU-X also provides information about the system's memory (RAM), including its size, type, and timings. Furthermore, it can detect the graphics card and display its essential details. The tool is useful for system administrators, hardware enthusiasts, and anyone who needs to quickly identify the components of a computer.
