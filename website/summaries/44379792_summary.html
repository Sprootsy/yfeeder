LM Studio is now an MCP Host

LM Studio v0.3.17 brings several enhancements and fixes to improve the user experience and functionality of the platform. The update includes a new 'Reset Chat' button in the Chat UI for clearing conversation history, providing a cleaner experience. The Model Download UI has been improved with more information displayed, like license types, and clarifies the functionality of the 'Show All Versions' button. This update also addresses issues related to incomplete model downloads and improves error handling. Additionally, the update introduces a 'Retry' button for failed downloads, allowing users to attempt the download again without restarting LM Studio. The update also includes fixes for specific issues on Windows, such as problems with AVX2 support, and general improvements to the stability and speed of the application. There are also fixes to the display of local models, and an update to the bundled version of ollama.

Key Points:

  • New 'Reset Chat' Button: Added to the Chat UI for clearing the conversation history.
  • Improved Model Download UI: Displays more information about models, including license types, and clarifies the functionality of the 'Show All Versions' button.
  • Fix for Incomplete Model Downloads: Addresses issues where model downloads were not completing correctly.
  • 'Retry' Button for Failed Downloads: Allows users to retry failed downloads without restarting LM Studio.
  • Windows AVX2 Issue Fix: Resolved problems related to AVX2 support on Windows.
  • Improved Application Stability and Speed: General improvements to the stability and speed of the application.
  • Fixes to Display of Local Models: Resolved problems with how local models were being displayed.
  • Updated ollama Version: The bundled version of ollama has been updated.
