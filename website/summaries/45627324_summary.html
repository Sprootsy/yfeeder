Ripgrep 15.0

Ripgrep 15.0.0 is a new major release that includes several new features, performance improvements, and bug fixes. A notable new feature is the addition of an API to the `grep-printer` crate, which enables the development of tools that can emulate ripgrep's output format.

A new option, `--no-messages`, has been added to suppress error messages. A new flag, `--json-truncate`, allows to truncate long lines in JSON output.

Performance improvements include reduced allocation overhead and faster handling of UTF-16 input on Windows.

Several bugs have been fixed, including issues with handling Unicode codepoints and incorrect output when using the `-o` flag with lookarounds.

The release also includes updates to dependencies and improvements to the build process.

Key Points:

  • New API in `grep-printer` crate for emulating ripgrep's output format.
  • Added `--no-messages` option to suppress error messages.
  • Added `--json-truncate` flag to truncate long lines in JSON output.
  • Reduced allocation overhead and faster UTF-16 handling on Windows.
  • Bug fixes for Unicode codepoint handling and `-o` flag with lookarounds.
  • Dependency updates and build improvements.
