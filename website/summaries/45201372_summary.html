Delphi 13 Florence Released

RAD Studio 13 Florence is now available. This release focuses on developer productivity and includes features designed to improve the development experience across various platforms.

Key enhancements in RAD Studio 13 Florence include improved code insight with Language Server Protocol (LSP) support, which provides more accurate and faster code completion, navigation, and refactoring. The LSP support is extended to cover more of the Delphi language and includes support for both Delphi and C++Builder.

The release also introduces enhanced debugging capabilities. Debugging is improved across platforms. Enhancements include improved call stack information and breakpoint management.

RAD Studio 13 Florence includes enhanced support for the latest operating systems and platforms, ensuring applications built with RAD Studio can target a wide range of devices. RAD Studio 13 Florence continues to support cross-platform development, allowing developers to write code once and deploy it to multiple platforms, including Windows, macOS, iOS, Android, and Linux.
