Kevo app shutdown

Summary:

The Kevo app is being shut down on February 29, 2024. This means that users will no longer be able to use the app to operate their Kevo smart locks. Specifically, features like unlocking/locking the door, sharing eKeys, and customizing lock settings through the app will no longer be available. However, the Kevo lock will continue to function using the key fob and standard key. Users are advised to transition to using the key fob or standard key for access after the app shutdown.

Key Points:

  • The Kevo app will no longer function after February 29, 2024.
  • App features like unlocking/locking, eKey sharing, and settings customization will cease.
  • The Kevo lock will still operate using the key fob and physical key.
  • Users should switch to using the key fob or physical key for continued access.
