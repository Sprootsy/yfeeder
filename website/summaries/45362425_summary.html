Zed's Pricing Has Changed: LLM Usage Is Now Token-Based

Summary:

The Zed editor has announced a change to its pricing model for LLM (Large Language Model) usage, moving from a per-user, per-month flat rate to a token-based consumption model. Previously, Zed offered unlimited LLM usage for $25 per user per month. However, due to the high costs associated with running LLMs, particularly with the increasing demand and generous usage by users, the previous model was unsustainable.

The new token-based system aims to align costs more closely with actual usage. Users will now be charged based on the number of tokens processed by the LLM. The announcement emphasizes that this change is necessary to ensure the long-term viability of offering LLM features within Zed. The new pricing takes effect on June 1, 2024.

Zed assures users that they are committed to providing a transparent and predictable pricing structure. Users will be able to monitor their LLM usage and associated costs through a dashboard. They also state that they will provide tools and guidance to help users optimize their usage and manage their spending.

The blog post also highlights Zed's intention to remain competitive and offer more affordable LLM usage compared to other services. They will achieve this through techniques such as model optimization and caching. Users who do not use LLM features will not be affected by this pricing change.

Furthermore, Zed plans to introduce a system that enables users to bring their own API keys from LLM providers, giving them even greater control over pricing and model selection. This feature is expected to be available in the future.

Key Points:

  • Zed is changing its LLM pricing model from a flat monthly rate to a token-based consumption model.
  • The previous unlimited usage model was unsustainable due to high LLM costs.
  • The new pricing goes into effect on June 1, 2024.
  • Users will be charged based on the number of tokens processed.
  • Zed will provide tools for monitoring usage and managing costs.
  • Zed aims to offer competitive pricing through optimization and caching.
  • Users who do not use LLM features will not be affected.
  • Zed plans to allow users to bring their own LLM API keys in the future.
