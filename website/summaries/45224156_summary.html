QGIS is a free, open-source, cross platform geographical information system

QGIS is a free and open-source Geographic Information System (GIS) application. It allows users to create, edit, visualize, analyze, and publish geospatial information. QGIS is cross-platform, running on Linux, Unix, macOS, and Windows. It supports a variety of vector, raster, and database formats. QGIS provides a plugin architecture that enables users to extend its functionality, and a Python API for scripting.

Key points:

  • QGIS is a free and open-source Geographic Information System (GIS).
  • It can be used to create, edit, visualize, analyze and publish geospatial information.
  • QGIS is cross-platform.
  • It supports vector, raster and database formats.
  • It has a plugin architecture.
  • It has a Python API.
