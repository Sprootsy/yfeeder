Solving Fizz Buzz with Cosines

The article explores an unusual solution to the classic Fizz Buzz programming problem, using cosine functions. The traditional Fizz Buzz challenge involves iterating through a range of numbers and printing "Fizz" for multiples of 3, "Buzz" for multiples of 5, and "FizzBuzz" for multiples of both. Numbers not divisible by either 3 or 5 are printed as they are.

The author presents a method that avoids conditional statements (if-else) and modulo operations, which are typically used to determine divisibility. Instead, it leverages the properties of cosine waves and their behavior around multiples of π.

The core idea is that cos(nπ) equals 1 when n is an even integer, -1 when n is an odd integer, and a value between -1 and 1 otherwise. The approach defines two functions based on cosine:

  • f(n) = cos(nπ/3). If n is a multiple of 3, then nπ/3 is a multiple of π. If n is even, f(n) is 1. If n is odd, f(n) is -1.
  • g(n) = cos(nπ/5). If n is a multiple of 5, then nπ/5 is a multiple of π. If n is even, g(n) is 1. If n is odd, g(n) is -1.

Then, the article uses these properties to construct expressions that evaluate to specific values based on divisibility by 3 and 5. It introduces the following:

  • Fizz condition: abs(cos(nπ/3)) = 1 if n is a multiple of 3 and not zero.
  • Buzz condition: abs(cos(nπ/5)) = 1 if n is a multiple of 5 and not zero.

The author ingeniously uses these cosine-based conditions to create strings "Fizz" and "Buzz" that are either present or empty depending on whether the number meets the divisibility criteria. These strings are then concatenated to produce the desired output. When neither "Fizz" nor "Buzz" is generated, the original number is printed.

The author then moves on to show how the solution could be golfed, meaning shortened.
